# AWS Cloud Labs

This repository contains hands-on AWS cloud infrastructure labs focused on EC2 deployment, Linux server administration, load balancing, and Nginx web hosting.

## Technologies Used

* AWS EC2
* Linux
* Nginx
* Load Balancing
* Networking
* Security Groups
* Bash/Linux Commands

## Labs Included

* AWS EC2 Web Server Deployment
* AWS Load Balancer Configuration
* Nginx Web Server Setup
* Linux Server Administration

## Skills Demonstrated

* Deploying and managing EC2 instances
* Configuring Linux-based cloud servers
* Verifying web server connectivity
* Basic load balancing concepts
* Troubleshooting infrastructure and networking issues
* Working with public IP addresses and web services
